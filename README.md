# Signite — Thai Sign Language Web App

## โครงสร้างโปรเจกต์
```
signite/
├── index.html              ← หน้าหลัก
├── vocabset.html           ← หน้าเลือกชุดคำศัพท์
├── practice.html           ← หน้าฝึกท่าทาง
├── CSS/
│   ├── main.css            ← สไตล์หน้าหลัก
│   └── style.css           ← สไตล์ vocabset + practice
├── JS/
│   └── app.js              ← logic การจับท่าทาง
├── IMG/
│   ├── Logo_signite.svg
│   ├── home_bg.svg
│   └── thumbnail_vocabset1.svg
└── VDO/
    ├── zero.mp4 ~ nine.mp4
    ├── chan.mp4, rak.mp4
    ├── khobkhun.mp4, maipenrai.mp4, sabaidee.mp4
```

## วิธีรัน
1. เปิด VS Code → File → Open Folder → เลือกโฟลเดอร์ signite
2. คลิกขวาที่ index.html → Open with Live Server
3. เว็บจะเปิดที่ http://127.0.0.1:5500
